2012款Retina macbook pro扩容廉价升级方案

发表于 2017-03-27 | 更新于 2017-03-27 | 分类于 硬件 | | 阅读次数

前言：

此教程适用于图中 可“代用”SSD的mac。

本人就是mc975 第一代15寸 Retina MBP，
也是很老的mac了，我这是标配i7 三代+16G内存定制 + 256G SSD（sata通道），就算是现在也可以干掉不少中低端笔记本。现在就是感觉256G 的硬盘空间有点小。就想升级下硬盘还能再战3年。所以google 百度后找到以下几个方案。
目前的mac 扩容升级方案：

  1. 更换大容量原装SSD，
  2. 创见的兼容SSD ，需要手动开启Trim
  3. 转接板（分NGFF／msata）+ 普通NGFF／msata固态 ，手动开启Trim。只有2012款的mac Retina可以，因为他是走的SATA通道。以后的mac 固态都是走PCIe 通道了，速度当然也提高很多了。以下某宝的图，可代用就是可转接的
  1. TF卡 + Nifty MiniDrive，读写速度有限
  2. NAS ，时光胶囊

个人认为方案3是性价比最高的

  1. 1.2.3其实都是更换SSD，当然难度相差不大，读写速度也和原来相差不大（速度受限于SSD本身和SATA接口速度，与转接板无关），当然也是动手比较多的。
  2. 4 是最简单的快捷的，但是速度受限于TF卡速度。
    1. 使用NAS，时光胶囊等实现云扩容。这个看个人了，其实成本也可高可低，你可以自己组装黑群晖，也可以买白群晖或者时光胶囊，这当然成本不低，当然也可以自己搞一个支持openwrt的路由+移动硬盘。这个方案使用就有地区限制或者速度限制。
      我把原装256G 固态卖了700，硬盘盒已经有，SSD送转接板（转SATA）
      所以一共花费50多完成了256G 升级 512G

      最后成果

      我是MC975 第一代15寸 rMBP。
      原来原装的SSD 256G

      更换为转接板+普通NGFF 512G SSD，几乎SATA 3 6Gb／s速度极限了。

准备

  1. 转接板（NGFF转SATA）+ 硬盘盒（SATA转USB）
  2. 转接板（NGFF转成 mac专用固态接口）
  3. SSD，NGFF 。注意NGFF不要选择2280长度和NVme固态。NGFF固态金手指一般为双开口
  4. 拆机工具P5（苹果专用） 和 T5 ，十字
    PS: 也可以使用msata 的固态，当然转接板要对应
    花费

拆机安装

进入Recovery模式系统 直接把系统盘恢复到新SSD里（需要NGFF转SATA再转USB）。没有硬盘，mac没法进入任一系统，包括恢复模式系统。
然后拆机。⚠️：拆固态 先把电池排线拆下。
拆机过程见 https://zh.ifixit.com/Device/Mac_Laptop
安装成果

装上直接可以开机。

开启trim

  1. 可以 安装 trim enable
  2. 也可以直接 终端
